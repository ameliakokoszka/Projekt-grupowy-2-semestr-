TEMAT PROJEKTU: Kalkulator Punktów Studenckich

OPIS: 

Aplikacja pomaga studentom śledzić wyniki z przedmiotów w aktualnym semestrze. Użytkownik dodaje przedmioty, ustawia progi punktowe dla ocen i na bieżąco wpisuje zdobyte punkty. Na tej podstawie aplikacja wyświetla aktualną ocenę oraz informuje, ile punktów brakuje do uzyskania kolejnej.

Aplikacja jest przeznaczona dla studentów, których przedmioty oceniane są punktowo (np. kolokwia, projekty, laboratoria lub egzaminy). Wybraliśmy ten projekt, ponieważ ułatwia kontrolowanie postępów w nauce i daje funkcję, której obecnie nie oferuje USOS.

Opis aplikacji i repozytorium zostały już przygotowane. W kolejnych etapach zaplanujemy szczegółowe funkcjonalności i wygląd interfejsu, następnie zaimplementujemy główne funkcje aplikacji, a na końcu przeprowadzimy testy i przygotujemy wersję końcową.

Przy tworzeniu aplikacji planujemy korzystać z biblioteki PySide6.
